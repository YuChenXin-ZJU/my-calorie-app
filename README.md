# My Calorie App

AI智能卡路里计算器 - 使用人工智能分析食物图片并计算营养信息

## 功能特点

- 🤖 AI图像识别技术
- 📊 精确营养分析
- 🎨 现代化UI设计
- 📱 响应式设计

## 部署

本项目已配置为可直接部署到 Vercel。

### 环境变量

需要在 Vercel 中设置以下环境变量：

- `QWEN_API_KEY`: 通义千问API密钥

## 本地开发

```bash
npm install
npm run dev
```

## 技术栈

- Frontend: HTML5, CSS3, JavaScript
- Backend: Node.js, Vercel Serverless Functions
- AI API: 阿里云通义千问 (Qwen)
